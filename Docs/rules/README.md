# Rules

## General Server Rules

```
1. No blank nicknames
2. No inappropriate nicknames:
  2.1 No sexually explicit nicknames
  2.2 No offensive nicknames
  2.3 No nicknames with unusual or unreadable Unicode
3. No inappropriate profile pictures:
  3.1 No sexually explicit profile pictures
  3.2 No offensive profile pictures
4. Moderation rules:
  4.1 Moderators don't reserve the right to change nicknames
  4.2 Moderators reserve the right to use their own discretion regardless of any rule
5. Loop holes / glitches:
  5.1 No exploiting loopholes in the rules (please report them)
  5.2 No random DMing other members of the server
  5.3 No bugs, exploits, glitches, hacks, bugs, etc.
6. Administration rules:
  6.1 No inviting unofficial bots
```
